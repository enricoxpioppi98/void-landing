# VOID — Black Hole Data Visualization Platform

## What we're building
25 landing page iterations for **VOID**, a fictional black-hole data visualization platform.
Plus 1 gallery page (index.html) that tells the story of the design process.

## Aesthetic
- Inspired by Arrival and Interstellar
- Extreme contrast, dark voids, gravitational warping
- Concentric rings, data-as-art
- Sleek, tech-y, elegant
- Tagline: "See beyond the event horizon."

## Tech constraints
- Pure HTML + CSS only — no JavaScript, no frameworks, no external APIs
- Each version lives in its own folder (v1/, v2/, etc.) with index.html + style.css
- Gallery page at root index.html

## Style preferences
- Dark backgrounds (#000 or near-black)
- Monospace and elegant serif fonts (system fonts + Google Fonts via link)
- Subtle CSS animations where appropriate (no JS)
- Minimal color palette — whites, grays, with occasional accent
